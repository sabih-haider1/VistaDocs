# VistaDocs Center

Complete Visa & Document Solutions website built with Node.js, Express, and EJS.

## Installation

```bash
npm install
```

## Run the Application

```bash
npm start
```

The application will run on `http://localhost:3000`

## Project Structure

```
VistaDocs/
├── app.js                 # Main Express server
├── package.json          # Dependencies
├── public/               # Static assets
│   ├── css/
│   │   └── style.css    # Main stylesheet
│   └── img/             # Images folder
└── views/               # EJS templates
    ├── partials/        # Reusable components
    │   ├── header.ejs
    │   ├── footer.ejs
    │   └── layout.ejs
    ├── home.ejs
    ├── about.ejs
    ├── services.ejs
    └── contact.ejs
```

## Features

- Responsive design
- Clean, modern UI with teal and white theme
- Express routing for all pages
- EJS templating with partials
- Professional government-service aesthetic
